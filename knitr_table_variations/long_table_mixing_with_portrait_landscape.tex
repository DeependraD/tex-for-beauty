\subsection{Page display modes: portrait and
landscape}\label{page-display-modes-portrait-and-landscape}

\begin{itemize}
\tightlist
\item
  to get every pages in landscape mode, add:
  \texttt{classoption:\ landscape} to the yaml header.
\item
  \texttt{\textbackslash{}blandscape} and
  \texttt{\textbackslash{}elandscape} commands come from modifications
  of command in pdflscape package.
\item
  use \LaTeX~\texttt{pdflscape} package instead of \texttt{lscape}
  package to get automatically rotated view of landscape pages in pdf
  viewer.
\end{itemize}
